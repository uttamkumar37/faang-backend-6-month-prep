# DSA Problem Log

Track every problem you attempt. Goal: **200+ problems** in 6 months with revision history.

## How to use

1. Add a row for every problem you attempt (first pass or revision).
2. Mark `Status` as: `✅ Solved` | `⚠️ Optimized` | `❌ Failed` | `🔁 Revisit`
3. Set `Difficulty`: `Easy` | `Medium` | `Hard`
4. Record your honest time taken.
5. Every Sunday: re-attempt all `🔁 Revisit` rows.
