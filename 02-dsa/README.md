## FAANG difficulty target

- Easy: should be automatic, under 10 minutes.
- Medium: 20-30 minutes, clean code, edge cases covered.
- Hard: should be discussable with structured approach, partially solvable under pressure.

## Practice cadence

- 2 problems per day, 5 days a week.
- Saturday: re-solve the 3 hardest problems from the week.
- Sunday: 1 timed mock (60 minutes, 2 problems).

## How to navigate quickly

1. Start with the numbered note in the relevant folder.
2. Work through `questions.md` before reading the implementation.
3. Compare your answer with the Java file.
4. Run or read the available test file when the folder includes one.
